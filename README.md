# Priya Patel — Static Portfolio Website

This is a static portfolio website for Priya Patel.

## Project structure

```text
portfolio/
  index.html
  styles.css
  script.js
  assets/
    css/
      styles.css
    js/
      script.js
    img/
      favicon.svg
  README.md
```

## Run locally

```powershell
cd "c:\Users\DELL\OneDrive\Desktop\portfolio"
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Edit

- `index.html` for content and sections
- `styles.css` for main theme and styling source
- `script.js` for main animations and interactions source
- `assets/css/styles.css` and `assets/js/script.js` are structured entry paths used by the page
